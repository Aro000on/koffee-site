# ☕ Koffee - Minimalist Coffee Site
**Примечание:** Koffee - это не опечатка, а название. Именно так задумано.
![Image] (https://meta.iaron.dev/koffee/og-image.jpg) 
### О проекте
Простой и чистый лендинг кофейни. 
---

### Что внутри
* **Стиль:** Минимализм. Никакого визуального мусора.
* **Адаптив:** Работает на мобилках. Есть меню-бургер на чистом CSS.


---

### Технические детали
1.  **Графика:** * Логотип и иконки - только `SVG`.
    * Фотки в `WebP` (для скорости) и `JPEG` (для надежности). 
    * Вес каждой картинки - до 200 КБ (сжато на совесть).
2.  **Верстка:** * Использован тег `<picture>` для правильной подгрузки форматов.
    * Сетки на Flexbox и Grid.
    * Никаких тяжелых библиотек, только чистый HTML/CSS.

---

### Как запустить
Просто открываешь `index.html` в любом браузере.
